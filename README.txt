Sådan sætter du op til brug af applikationen:
1:  Inde i mappen "DbContext" i filen "DBContext.cs" ligger connectionstring. Tilpas her Password så det passer til din SQL Server. 
2:  Start din SQL Server.
3:  Brug "Package Manager Console" og skriv "Update-Database" i konsolvinduet/ Hvis terminalen benyttes i stedet, kan CLI commandoen: "dotnet ef database update" benyttes i stedet. 
4:  Kør konsol applikationen ved med "Ctrl + F5".

Der er læsevejledning undervejs i applikationen.
Brug store bogstaver til styring af applikationen.
